# Delete Widget

For permissions, refer **[Dashboard Views](https://help.5day.io/docs/user-guide/Analytics/Dashboard%20View)**

To delete an existing widget:

1.  Navigate to the **Dashboard View**.
2.  Click the **Edit Dashboard** button located at the top right corner. (This will enable dashboard editing mode)
3.  Hover over the widget you wish to delete - you will see a **Delete** option at the top right.
4.  Click on **Delete**.
5.  A **confirmation screen** will appear for you to confirm the deletion.
6.  Upon confirmation, the widget will be **permanently deleted** from the Dashboard.
