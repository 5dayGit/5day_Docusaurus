# Move Widget
The Move Widget feature allows you to reposition widgets on your dashboard based on your layout preferences.

To move a widget:

1.  Enable **Edit** mode by clicking the **Edit Dashboard** option.
2.  Hover over the widget you want to move.
3.  Drag the widget to your desired position on the dashboard.
4.  Release the widget to place it. The new position will be updated and **auto-saved** instantly.
