# Edit Widget

For permissions, refer **[Dashboard Views](https://help.5day.io/docs/user-guide/Analytics/Dashboard%20View)**

To edit an existing widget:

1.  Locate the widget you want to modify on the **Dashboard**.
2.  **Hover over** the widget - at the top-right corner, you will see a **‘Manage’** option.
3.  Click **‘Manage’** to open the widget's configuration settings.
4.  Make the necessary changes to the configuration.
5.  After editing, click on **‘Update Widget’** to save your changes.

✅ The widget will immediately update on the Dashboard based on the new settings.
