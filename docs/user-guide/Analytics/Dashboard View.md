## Permission Levels
Access to create, edit, or delete widgets in the Project/Section Dashboard is based on project-level permissions.

| Project Permission Level | Description |
|--|--|
| Edit - None  | User **cannot** configure any widget for the project/section. |
| Edit - Owned | User can manage the dashboard **only for projects** where they are the **Project Owner**. |
| Edit - Related  | User can manage the dashboard **for projects** where they are either the **Project Owner** or a **Project Member**. |
| Edit - All | User can manage the dashboard **for all projects** in the workspace. |

âœ… Users with relevant permissions can add, modify, or remove widgets within the dashboard to customize their project view as needed.

## Enable Dashboard View

By default, in all projects & section; dashboard view is enable.
Follow these simple steps to manage the visibility of dashboard view in project/section views panel.

 1. Navigate to the specific **Project -> Settings** from the left sidebar. 
 2. Select the **Views**. 
 3. Locate **Dashboard View**.
 4. You can enable/disable view. 
 5. You can also change the order of the view to display in project views panel.

## Access Dashboard View

Follow these simple steps to access and manage the Dashboard in 5day.io.

1.  Navigate to the **Projects** list from the left sidebar.
2.  Select the specific **Project/Section**  you want to open.
3.  In the project/section views, **switch to "Dashboard" view** using the view selector.
4.  You can now **add, edit, or delete widgets** within the project or section dashboard based on your permissions.
