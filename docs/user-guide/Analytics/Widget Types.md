# Widget Types

Widgets in 5day.io are categorized based on the type of data you visualize.
 - Tasks
 - Time Entries
 - Sprints
   
Each widget type helps you to quickly monitor different aspects of project performance and team productivity.  

Depending on what information you want to analyze, you can choose the appropriate widget type and customize its display.  

This structure ensures that you can access actionable insights in an intuitive, organized manner.

## Task Widgets
Task Widgets allow you to visualize the status, progress, and distribution of tasks within a project or section.  

These widgets help you track overdue tasks, monitor task priorities, view completion rates, and many more.  

You can use various chart types like Pie Charts, Bar Charts, Line Charts, doughnut Charts, Number Charts and Table views.

Bar Chart for Tasks: Displays task data using horizontal or vertical bars to easily compare quantities across different categories.

Line Chart for Tasks: Shows task trends over time by connecting data points with a continuous line, ideal for tracking progress.

Pie Chart for Tasks: Represents task distribution as slices of a circle, helping visualize proportions of different statuses or priorities.

Doughnut Chart for Tasks: A circular chart similar to a pie chart, but with a hollow center, offering a cleaner look for task breakdowns.

Number Chart for Tasks: Highlights a key metric as a single, bold number â€” like total tasks, completed tasks, or overdue tasks.

Table view for Tasks: Organizes task details in a structured table format for easy sorting, filtering, and comparison.

## Time Entry Widgets

Timesheet Widgets focus on time tracking and work hours logged by team members.  

These widgets help you monitor billable hours, non-billable efforts, and overall team utilization effectively.  

Popular chart types for Timesheet Widgets include Bar Charts, Pie Charts, and Summary Tables.

Bar Chart for Time Entries - A visual representation of time entries using horizontal or vertical bars to compare different time data.

Line Chart for Time Entries - A chart that displays time entries as data points connected by lines, showing trends over a period.

Pie Chart for Time Entries - A circular chart divided into sectors representing the proportion of time entries relative to a whole.

Doughnut Chart for Time Entries - A variation of the pie chart with a central hole, used to display the percentage distribution of time entries.

Number Chart for Time Entries - A chart that shows the total or specific numerical value of time entries, typically represented in large numbers.

Table view for Time Entries - A tabular representation of time entries with rows and columns, providing detailed data in a structured format.

## Sprint Widgets

Sprint Widgets are designed to track sprint progress, team velocity, and sprint goals for agile teams.  

These widgets are ideal for monitoring real-time agile delivery performance during a sprint.  

There are three key Sprint Widget types available:

Sprint Burn Up Chart: Shows completed work compared to the total scope.

Sprint Burn Down Chart: Displays remaining work over the sprint timeline.

Velocity Chart: Visualizes the amount of work completed sprint-over-sprint.
