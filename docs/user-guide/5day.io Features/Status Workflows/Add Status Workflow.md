## Overview
A Status Workflow defines the set of statuses your team uses to track task progress from start to finish.

You create the status workflow at the **Workspace level**, then add the statuses you want under each stage.
## **How to add a Status Workflow**
1. Log in to [**5day.io**](http://5day.io/).
1. Go to the workspace where you want to create the workflow.
1. Click the **three-dot (⋮) icon** next to the workspace name.
1. Click **Workspace Settings**.
1. From the left menu, click **Customization**.
1. Click **Status Workflow**.
1. Click **+ Add** (top-right).
1. Enter a **Status Workflow name**.
1. Click **Save**.

If you do not want to create it, click **Discard**.

**Note:** You can add statuses only after the workflow is created and saved.

## **How to add statuses to a Status Workflow**
1. Select the workflow from the left list.
1. Under **New**, **In Progress**, or **Done**, click **+ Add Status**.
1. In the dropdown: 
   1. **Search and select** an existing status, **or**
   1. Type a new status name and click **+ Create New Status**
1. Repeat the same steps to add more statuses under each stage.

**Note:** When you click **+ Create New Status**, the status is created instantly if it does not already exist.
