## Overview

In [5day.io](http://5day.io/), you can import tasks directly into a specific project using a structured Excel template.

This method allows you to upload tasks in bulk without navigating to Account Settings.

It is useful when adding multiple tasks to a single project.

The import process uses the same Excel template and validation logic as Account-Level Import.

Only the navigation path is different.
## **How to Import Tasks**
1. Log in to [5day.io](http://5day.io/).
1. Go to the workspace where your project exists.
1. Open the required **Project**.
1. Click the **three-dot menu** next to the project name.
1. Click **Import**.
1. In the Import popup, ensure **Task** is selected as the import type.
1. Click **Download Sample File**.

The Excel file contains two sheets:

- **Template** – Enter your task data (Task Name, Description, Due Date, Assignee, etc.).
- **Guidelines** – Reference instructions for required formats and valid values.
1. Open the downloaded file.
1. Enter your task data in the **Template** sheet.
1. Save the file.
1. Return to the Import popup in [5day.io](http://5day.io/).
1. Click **Upload from computer**.
1. Select your completed Excel file.
1. Click **Import**.
## **After the Import**
Once the file is processed:

- The system validates all records.
- Tasks are created in the selected project.

If any records fail validation:

- The system displays the number of failed records.
- You can download the error file.
- Correct the issues in your Excel file.
- Re-upload the corrected file.
