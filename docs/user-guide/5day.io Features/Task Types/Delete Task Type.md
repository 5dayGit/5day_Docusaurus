## Delete Task Type
You can delete a Task Type from the Workspace if it is no longer required.

⚠ **Important:**

A Task Type cannot be deleted if it is currently mapped to any project.

You must first remove it from all mapped projects before deleting it from the Workspace.
## **How to Delete a Task Type**
1. Log in to [**5day.io**](http://5day.io/).
1. Navigate to the **Workspace** where the Task Type exists.
1. Click the **three-dot (⋯) menu** next to the Workspace name.
1. Click **Settings**.
1. From the left panel, click **Customization**.
1. Click **Task Type**.
1. From the list, locate the Task Type you want to delete.
1. Click the **three-dot (⋯) menu** next to that Task Type.
1. Click **Delete**.
1. A confirmation dialog will appear.
1. Click **Yes** to confirm.

If the Task Type is not mapped to any project, it will be deleted immediately.
## **If Deletion is Blocked**
If the Task Type is mapped to a project, you will see an error message indicating that it cannot be deleted.

To proceed:

1. Go to the **Project** where the Task Type is mapped.
1. Navigate to: 
   1. **Project Settings**
   1. **Customization**
   1. **Task Type**
1. Remove the Task Type from that Project.
1. Repeat this step for all projects where it is mapped.
1. Once removed from all projects, return to: 
   1. **Workspace Settings → Customization → Task Type**
1. Delete the Task Type from the Workspace.
