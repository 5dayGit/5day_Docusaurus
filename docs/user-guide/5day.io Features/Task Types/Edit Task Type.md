## **Overview**
You can edit an existing Task Type at the workspace level.

Editing a Task Type allows you to:

- Rename the Task Type
- Change its icon
- Change its color
- Update its Status Workflow
- Add or modify Custom Fields

All changes apply immediately across the workspace.
## **How to Edit a Task Type**
1. Log in to [5day.io](http://5day.io/).
1. Go to the workspace where the Task Type exists.
1. Click the **three-dot (⋮) menu** next to the workspace name.
1. Click **Settings**.
1. From the left panel, click **Customization**.
1. Click the **Task Type** tab.
1. From the Task Type list, click the Task Type you want to edit.
1. Update the required details (name, icon, color, workflow, or custom fields).
1. Make the required changes.

Changes are saved automatically.
## **Important Notes**
- Changes to Task Type name are reflected across all projects immediately.
- Existing tasks are updated automatically when name is changed.
- Changing Status Workflow requires manual status remapping if tasks already exist.
- Task Type can be edited at Workspace level only.
