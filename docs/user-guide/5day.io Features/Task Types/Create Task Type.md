## Overview
Task Types are created at the **workspace level**.

A Task Type helps you standardize how a specific kind of work should run by allowing you to configure:

- A **Status Workflow**
- **Custom Fields**

After you create a Task Type, you can select it while creating tasks in projects.
## **How to Create a Task Type**
1. Log in to [5day.io](http://5day.io/).
1. From the left panel, go to the workspace where you want to create the Task Type.
1. Click the **three-dot (⋮)** menu next to the workspace name.
1. Click **Settings**.
1. From the left panel, click **Customization**.
1. Click **Task Type**.
1. Click **+ Add**.
1. Enter a **Task Type name** (example: *Bug*).
1. (Optional) Select an **icon** and **color**.
1. Click **outside the name field** to save.

✅ The Task Type is created.
## **Notes**
- Status workflow assignment is not mandatory during creation.
- A **default workflow** is assigned automatically after creation.
- You can set a Task Type as **Default** only after it is created.
- After creating a Task Type, you can click it to: 
  - Change the **Status Workflow**
  - Add **Custom Fields**
