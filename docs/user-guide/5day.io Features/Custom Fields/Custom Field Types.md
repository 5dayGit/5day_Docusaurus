# Custom Field Types

The availability of custom field types may vary based on your subscription.

5day.io offers below custom field types:

| Field Type | Description |
|--|--|
| Currency| Allow you to input and display monetary values with automatic currency symbols and formatting, ideal for tracking budgets, expenses, or financial metrics in projects and tasks. |
| Date | Capture dates, which can be useful for tracking deadlines, milestones, or other time-sensitive information. |
| Date & Time | Allows you to specify both the date and time for a particular event or deadline, offering a high level of precision in scheduling. |
| Dropdown | Provide a predefined list of options for users to choose from, ensuring consistency in data entry. |
| Email | To store and manage email addresses relevant to tasks, projects, or team members. |
| Multiselect Dropdown | Provide a predefined list of options for users to select multiple values, ensuring consistency in data entry. |
| Number| Ideal for storing numerical data, such as cost estimates or quantities. |
| Percentage| Capture values as a proportion (e.g., 25%), perfect for progress indicators, completion rates, or allocation breakdowns in tasks.|
| Radio Button | Offer a binary choice, useful for tracking yes/no or true/false information. |
| Rating | To assign and track qualitative ratings to task, subtask and projects. It provides a subjective measure to express opinions, priorities, or levels of satisfaction. |
| Rich Text| Provides a formatted editor for multi-line content, supporting bold, italics, lists, links, images, and embedsâ€”great for detailed notes, requirements, or specifications. |
| Textbox | Allow users to input and store alphanumeric characters. |
| Text Area | To capture, store and display extended and detailed text-based information. |
| Time | Capture time, which can be useful for tracking durations, deadlines, or scheduled time intervals. |
| URL | To include and link external web addresses or URLs. |
