# Custom Fields
Custom fields are additional data fields that you can create and define based on your project's requirements.

These fields can store various types of information, such as text, numbers, dates, checkboxes, and more. You can choose where to place these fields in 5day.

Custom fields are stored at account level and can be used at multiple places in entire account. Custom Fields can be used to customize projects, tasks & subtasks.

## Why is Custom Field Important

Custom fields are your secret weapon for **tailoring 5day.io to your exact workflow** - no more forcing square pegs into round holes.

## Key Features of Custom Fields

-   **Flexible Types & Placement**: Choose from text, dates, dropdowns, numbers, URLs, and more - place fields in projects, tasks, or subtasks exactly where you need them.
-   **Central Library with Global Sync**: Create once in the **Custom Field Library**, use everywhere - edits update instantly across all projects and tasks.
-   **Granular Control**: Add fields at account, workspace, or project; make them required, filterable, or visible in views, boards, and reports.
-   **Seamless Integration**: Auto-sync with filters, group-by, list/board views, and analytics - your custom data flows effortlessly into planning and reporting.
