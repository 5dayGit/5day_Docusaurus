
# List of Custom Fields

To access all custom fields of an account

 1. Click User Icon (Bottom left panel) -> Account Settings -> Custom Fields
 2. List of all custom fields will appear; list will contain:
	 - Default custom fields offered by 5day.io.
	 - Custom fields created by your account members.

	 <img width="3560" height="2300" alt="image" src="https://github.com/user-attachments/assets/e2dfaec5-afe4-427d-9b58-3806bfeeac53" />


3. List will have below details associated with it
	 - Name of Custom Field
	 - Type of Custom Field
	 - Add Custom Field, Refer ‘[Add Custom Field](https://github.com/5dayGit/5day_Docusaurus/blob/main/docs/user-guide/5day.io%20Features/Custom%20Fields/Add%20Custom%20Field.md)’ for more details.
	 - Edit Custom Field, Refer ‘[Edit Custom Field](https://github.com/5dayGit/5day_Docusaurus/blob/main/docs/user-guide/5day.io%20Features/Custom%20Fields/Edit%20Custom%20Field.md)’ for more details.
	 - Delete Custom Field, Refer ‘[Delete Custom Field](https://github.com/5dayGit/5day_Docusaurus/blob/main/docs/user-guide/5day.io%20Features/Custom%20Fields/Delete%20Custom%20Field.md)’ for more details.
	 - Search Custom Field: Search any custom field via its 'Name'.
