# Upload File
Attach documents, images, or spreadsheets directly to a project, section, task, subtask, or message. 
Once uploaded, files stay linked to the work, ready for preview, download, or version updates.

## Subscription
The Files feature is available to all subscription plans. However, the total file upload limit depends on your plan:
| Subscription Plan | File Upload Limit |
|--|--|
| Free | 2 GB |
| Growth | 100 GB |

## Permission Set
Only users with upload permission can upload files to the Files View.

## Upload File

A file can be uploaded in three different ways:

**Upload from Computer**  
    Users can directly upload files from their local system. Select the file from your computer and click to upload.
    
**Upload via Google Drive**  
    You can upload files stored in your Google Drive. Simply connect your Google Drive account, select the file, and it will be uploaded to 5day.io.
    
**Upload via OneDrive**
    Upload files from OneDrive by connecting your OneDrive account. Select the file from OneDrive, and it will be uploaded to 5day.io.
