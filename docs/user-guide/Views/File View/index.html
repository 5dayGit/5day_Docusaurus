File View

Subscription​

The Files feature is available to all subscription plans. However, the total file upload limit depends on your plan:

Subscription PlanFile Upload Limit
Free2 GB
Growth100 GB

Availability of File view in Structure​

In 5day.io, files can be managed and organized across four levels of hierarchy:

  1. Project
  2. Section
  3. Task
  4. Subtask

At the Project and Section, the File View needs to be enabled in the Project Settings under the Views section.

At the Task and Subtask levels, the Files feature is automatically enabled.

Enabling File view​

By default, in all projects & section; file view is enable. Follow these simple steps to manage the visibility of file view in project/section views panel.

  1. Navigate to the Project Settings.
  2. Under the Views section, ensure that the File View is enabled.
  3. Once enabled, discussions will be available at the Project and Section.
  4. For Task and Subtask, the files feature is auto-enabled and will be accessible without additional setup.
  5. You can also change the order of the view to display in project views panel.

Who Can Access​

View Permission: Users with view access to the files can view them at all views (Project, Section, Task, Subtask).

How to Access​

For Project/Section:

  1. Go to the specific Project or Section.
  2. Change the view to ‘Files’ to view all the uploaded files for that entity.

For Task/Subtask:

  1. Go to the specific Project/Section.
  2. Select the Task or Subtask.
  3. On the task details page, click on ‘Files’ under Task/Subtask details.
  4. All uploaded files related to that task or subtask will be visible here.
