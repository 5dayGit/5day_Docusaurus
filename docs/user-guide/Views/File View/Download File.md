Downloading a file in 5day.io is one click away - just hover over the file and hit the download icon. 

## Permission Set
Only users with file view permission can download files.

## Download File

To download a file

1.  Access the file by navigating to the Files View in the relevant section.
2.  Click on the file.
3.  The file will be automatically downloaded to your local machine.
