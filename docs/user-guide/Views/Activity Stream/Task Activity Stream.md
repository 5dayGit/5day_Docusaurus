# Task Activity Stream
** Task Activity History** is the complete, unfiltered log of every single thing that ever happened to a Task or Subtask - from creation to deletion - showing exactly who did what and when.

## Subscription
| Plan | History Maintained|
|--|--|
| Free | 1 day |
| Growth | 30 days |
| Scale | 180 days |

## Who can View 
All users with View Task permission can open the Activity History for both Tasks and Subtasks.

## Access Activity Stream 

 1. Open any Task → **Task Details** panel (right side)
 2. Click the **Activity Stream** tab - you’re now in the full history

## Activity Stream Features

### Activity Types
| Activity Type | Details |
|--|--|
| Task Created | Who created the Task/Subtask + timestamp |
| Task Edited| Every field change: Status, Assignee, Due Date, Priority, Tags, Progress, etc. |
| Moved | Section or Project change |
| Task Deleted | Who deleted + timestamp (Task or Subtask) |
| Workflow Actions | Auto-changes from automations |

### Filter by Person
Want to see only specific person's changes?

-   Click **Filter** -> Select a person 
-   Instantly shows just his/her updates (e.g., “Sarah updated Status Workflow”)
-   Clear filter -> see all data again

### Date Range Groups
Activity Stream auto-groups entries to keep it scannable:
| Group | Details |
|--|--|
| Today | Changes from midnight to now |
| Last 7 days | Changes from last week |
| Last 30 days | Changes from past week |
| Older than 30 days | Everything before that |
