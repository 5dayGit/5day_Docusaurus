# Project Activity Stream
Project Activity Stream is the complete timeline of every change made directly to your Project - from settings tweaks to team updates - giving you an instant snapshot of who did what, when.

## Subscription
| Plan | History Maintained|
|--|--|
| Free | 1 day |
| Growth | 30 days |
| Scale | 180 days |

## Enable Activity Stream

1.  Go to your Project -> More icon (three dots) → **Project Settings** → **Views**
2.  Toggle **Activity Stream** to **ON/Off** (it’s on by default)
3.  Go back to **Project Views** -> click **Activity Stream** tab.

## Who can View 
All users with Project access (View permission or higher) can see the Activity Stream of project.

## Activity Stream Features
### Activity Types
| Activity Type | Details |
|--|--|
| Project Details | Name, description, icon/color changes and other details of project |
| People Management | Users added/removed, role changes |
| Features and Views | Feature and views toggles |
| Customization | Task Types, Status Workflows, Priorities, Custom Fields, Sections |
| Workflows | Workflows created/edited/deleted |
| Integration | Connections to Slack, teams, etc. added/updated |



### Filter by Person
Want to see only specific person's changes?

-   Click **Filter** -> Select a person 
-   Instantly shows just his/her updates (e.g., “Sarah updated Status Workflow”)
-   Clear filter -> see all data again

### Date Range Groups
Activity Stream auto-groups entries to keep it scannable:
| Group | Details |
|--|--|
| Today | Changes from midnight to now |
| Last 7 days | Changes from last week |
| Last 30 days | Changes from past week |
| Older than 30 days | Everything before that |
