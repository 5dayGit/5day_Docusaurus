# Section Activity Stream

**Section Activity Stream** is the full timeline of every change made to a specific Section - from name tweaks to status shifts - so you always know who touched what and when inside this bucket of tasks.

## Subscription
| Plan | History Maintained|
|--|--|
| Free | 1 day |
| Growth | 30 days |
| Scale | 180 days |

## Enable Activity Stream

1.  Go to your Project -> More icon (three dots) -> **Project Settings** -> **Views**
2.  Toggle **Activity Stream** to **ON/Off** (it’s on by default)
3.  Go to **Section** -> Section Views -> click **Activity Stream** tab.

## Who can View 
All users with Project access (View permission or higher) can see the Activity Stream of project.

## Activity Stream Features
### Activity Types
| Activity Type | Details |
|--|--|
| Section Details | Name, description, icon/color changes |
| Status & Dates | Status (New/In Progress/Done), Start Date, Due Date updates |
| Order & Visibility | Drag-and-drop reordering, Hide/Show in hierarchy |

### Filter by Person
Want to see only specific person's changes?

-   Click **Filter** -> Select a person 
-   Instantly shows just his/her updates (e.g., “Sarah updated Status Workflow”)
-   Clear filter -> see all data again

### Date Range Groups
Activity Stream auto-groups entries to keep it scannable:
| Group | Details |
|--|--|
| Today | Changes from midnight to now |
| Last 7 days | Changes from last week |
| Last 30 days | Changes from past week |
| Older than 30 days | Everything before that |
