# Reply Message

A **reply** is a response to an existing message, creating a thread under the original message. This allows for more detailed conversations within the same context.

## Permission Set
To reply to a message, users must have permission to add message.

## Reply Message

1.  Hover over the message you would like to reply to.
2.  A **Reply** button will appear at the top of message.
3.  Click on the **Reply** button to open a text input box for the reply.
4.  Type your reply in the input box.
5.  Click on the **Save** button to add your reply.

The reply will appear below the original message, creating a thread for easy tracking of conversations.
