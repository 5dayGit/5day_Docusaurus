# Add Message

A **Message** in the Discussion View allows team members to share their thoughts, updates, or feedback regarding specific projects, sections, tasks, or subtasks.

Adding a message is a key method of collaborating and keeping communication transparent within 5day.io.

## Subscription

The ‘Discussion’ feature is available across all subscription plans in 5day.io.

## Permission Set

Users who have the **permission to add message** can participate in discussions and contribute to the conversation.

## Add Message

1.  Navigate to the relevant **Project**, **Section**, **Task**, or **Subtask**.
2.  Change the view to **Discussion**.
4.  Type your message in the **text input box** at the bottom of the discussion area.
5.  If you want to format your text, use the **Rich Text Editor** options.
6.  After typing your comment, click on the **Save** button to add the message to the discussion.

Your message will be immediately visible to everyone who has access to that discussion.
