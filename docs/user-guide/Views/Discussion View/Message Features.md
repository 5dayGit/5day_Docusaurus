# Message Features

5day.io offers multiple features to enhance message functionality and improve communication in discussions:

## **@Mention**

Mentions allow you to tag other users in your message, notifying them directly about the message. This is useful for drawing attention to specific people or bringing them into a conversation.

**@mention User**

1.  In the message text input box, type the **@** symbol.
2.  Start typing the name of the person you want to mention.
3.  A list of people will appear, allowing you to select the correct user by clicking on their name.
4.  Once the person is tagged, they will receive a **notification** that includes a link to the message, allowing them to directly access the conversation.

**Quick Note**
Users tagged in message will also be able to reply directly from the notification.

## **Tag Task**

Tagging of task/subtask allow a quick access from a message and also shows the reference.

**#Task/Subtask**

1.  In the message text input box, type the **#** symbol.
2.  Start typing the ID/title of task/subtask.
3.  A list of tasks/subtasks will appear as per your searched criteria.
4.  Select the task/subtask.

## **Attach File**

The **Attach File** feature allows you to add files to your message. This can include documents, images, spreadsheets, or any other type of file relevant to the message.

**Attach a File**

1.  Click on the **Attachment icon** (usually represented by a paperclip).
2.  Choose one of the following upload methods:

-   **Upload from Computer:** Select a file from your local system to upload.
-   **Upload via Google Drive:** Select a file from your Google Drive to upload.
-   **Upload via OneDrive:** Select a file from your OneDrive to upload.

4.  After the file is uploaded, it will appear as an attachment under your message.

## **Rich Text Editor**

The **Rich Text Editor** in 5day.io provides a powerful and flexible way for users to create well-formatted, professional content across the platform.  

Whether you are writing a task description, adding a discussion message, or updating a document, the Rich Text Editor offers an extensive range of formatting tools to enhance your writing experience.

Users can easily apply text formatting (such as bold, italics, underline), insert lists, quotes, and code blocks, and add advanced elements like links, videos, and emojis.  

Additionally, options like text direction, font size adjustment, background highlights, and cleaning formatting help maintain a clean and structured layout.

The editor supports both basic and advanced editing needs, making collaboration and communication within 5day.io more effective and visually organized.

### Access Rich Text Editor

1.  Whenever you are editing a text field (like Task Description or Discussion), the Rich Text Editor toolbar will be available on top.
2.  To access advanced features, click on the dropdown (`...`) to open the **Rich Text Menu**.
3.  Hover over each option to view a tooltip describing its functionality.

### List of Features

Below are the features available in the Rich Text Editor:
| Feature | Description |
|--|--|
| **Bold** | Make your selected text bold for emphasis. |
| **Italic** | Italicize your text to highlight or differentiate content. |
| **Underline** | Underline important parts of your text. |
| **Strikethrough** | Strike through text to show completed or obsolete items. |
| **Bullet List** | Create a bulleted list for better organization of points. |
| **Number List** | Create a numbered list for sequential information. |
| **Blockquote** | Add a quotation block to highlight key quotes or references. |
| **Code Block** | Insert a block specifically for code snippets or technical content. |
| **Subscript** | Add subscript text for chemical formulas or technical data (e.g., H₂O). |
| **Superscript** | Add superscript text for footnotes or mathematical notations (e.g., x²). |
| **Indent** | Indent text to the right to structure the document better. |
| **Outdent** | Remove an indent to adjust the text back towards the left. |
| **Text Direction** | Change the direction of text, useful for right-to-left languages. |
| **Font Size** | Adjust the font size (increase or decrease) of your text. |
| **Header** | Insert header styles (H1, H2, H3, etc.) to structure the content into sections. |
| **Link** | Insert a hyperlink to direct users to an external or internal page. |
| **Video** | Embed a video by adding a video link. |
| **Color** | Change the text color to emphasize specific words or sections. |
| **Background** | Highlight text by changing its background color. |
| **Clean** | Remove any formatting applied to selected text and return it to default style. |
| **Undo** | Revert your last action or change made in the editor. |
| **Redo** | Reapply an action that was undone. |
