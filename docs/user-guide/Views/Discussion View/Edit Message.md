# Edit Message
An **edit** option allows users to modify the content of a message they have already posted. 
Like deleting, only the user who added the message can edit it.

## Edit Message

1.  Hover over the message you wish to edit.
2.  Click on the **Edit** icon (usually represented by a pencil).
3.  Modify the text of your message.
4.  After making changes, click on the **Save** button to update the message.
