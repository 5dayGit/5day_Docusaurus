## Overview 
Views in 5day.io help you organize and visualize your project tasks in the way that works best for you and your team. 

Whether you're reviewing a long list of tasks or managing work through visual stages, 5day.io supports multiple powerful ways to interact with your project: 
* **List View** – See all tasks in a structured list format. 
* **Board View** – View tasks as cards and drag them across stages (like “New,” “In Progress,” and “Done”). 
* **Timesheet View** – Track time entries across days, weeks, or months. 
* **Dashboard** – Analyze work through prebuilt reports and widgets. 
* **Files View** – Store, upload, and manage project files. 
* **Discussion View** – Start and reply to conversations around the project. 
* **Activity Stream** – View a real-time feed of all project changes. 

These views are **project-specific** and tightly integrated with your workflows. Users with project settings access can customize views to suit their needs. 
 
### List View 
List View presents your tasks in a linear, table-like format — ideal for users who prefer structured lists or spreadsheets. 
**Key Features:** 
* **Search**: Find tasks quickly using keywords. 
* **Filter**: Narrow down tasks by assignee, status, priority, etc. 
* **Sort**: Organize tasks by field (e.g., name, due date, priority). 
* **Group By**: Group tasks by section, assignee, or status, etc. 
* **Customize**: Show or hide fields like estimated time, task type, etc. 

### Board View 
Board View displays your tasks as movable cards across vertical columns — great for visualizing workflow stages. 
**Key Features:** 
* **Drag-and-drop**: Move tasks across columns to show progress. 
* **Card layout**: Tasks appear as visual cards with key details. 
* **Status-based grouping**: Columns reflect task status or custom groupings. 


### Timesheet View 
Timesheet View lets users track and manage their work hours through a calendar interface. 
**Key Features:** 
* **Calendar views**: Toggle between Day, Week, and Month. 
* **Manager access**: Users with permission can view others' timesheets. 
* **Filters**: Filter by team member or date range. 

### Dashboard View 
Dashboard View provides a visual summary of your project through charts and analytics widgets. 
**Key Features:** 
* **Widgets**: Create and manage your own widgets to understand project progress at a glance. 


### Files View 
Files View is your central hub for managing documents related to a project. 
**Key Features:** 
* **Upload**: Add files directly to the project. 
* **Preview**: View files inside 5day.io without needing external tools. 
* **Organize**: Keep all relevant documents in one place. 

### Discussion View 
Discussion View allows team members to collaborate and share updates within a project. 
**Key Features:** 
* **Post messages**: Share comments to share updates or ask questions. 
* **Tag teammates**: Use @mentions to involve others in the conversation. 
* **Reply**: Continue threads in a structured format. 


### Activity Stream 
Activity Stream provides a detailed timeline of all actions that have occurred within a project. 
**Key Features:** 
* **Complete audit trail**: View who created the project, added users, changed settings, enabled views, and more. 
* **Chronological log**: Activities are listed in the order they occurred, offering a clear project history. 
* **Project-specific**: Only actions related to that particular project are shown here. 

### How to Enable or Disable Views in a Project 
In 5day.io, you can customize which views are available for your project, such as List, Board, Timesheet, Dashboard, Files, and more. 
**Steps to Enable or Disable Views:**
**1.** **Open the Project**   Navigate to the project where you want to update the views. 
**2.** **Access Project Settings**   Click the **three-dot menu** (︙) in the top-right corner of the project.   Select **Project Settings** from the dropdown. 
⠀✅ *Only users with permission to manage project settings will be able to modify views.* 
**3.** **Go to the Views Tab**   Inside Project Settings, click on the **Views** tab. 
**4.** **Configure Views** 
* You’ll see a list of available views like **Board**, **Timesheet**, **Dashboard**, **Files**, **Discussion**, and **Activity Stream**. 
* **List View** is always enabled and cannot be disabled. 
* Use the toggle switches next to each view to turn them on or off based on your preference. 
* You can also **reorder views** to set the preferred navigation order by dragging them up or down. 
**5**. **Changes are Saved Automatically**   Once you make your selections, the settings are applied instantly—no need to click Save.
