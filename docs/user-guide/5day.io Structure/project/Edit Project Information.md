### **How to Edit Project Information**
To edit the project info in 5day.io, follow these steps:

1. **Go to Project Settings**: First, navigate to the **Project Settings** of the project you want to edit.
1. **Access Project Info**: In the Project Settings menu, find and click on **Project Info**.
1. **Edit Project Info**:
   1. **Project Name**: You can change the project name by clicking on the project info icon.
   1. **Project Icon**: Upload a custom image or select pre-defined icon and colour/theme for the project icon.
   1. **Start and Due Dates**: Set the **start date** and **due date** for the project.
   1. **Project Status**: Choose the current **status** of the project (e.g., New, In Progress, In Review and Done).
   1. **Priority**: Set the **priority** level of the project (e.g., High, Medium, Low).
   1. **Project Prefix**: Select a **prefix** for the project, which will be used for task IDs.
   1. **Project Owner**: Assign the **owner** for the project. If there are multiple owners, click **Add Owner** and add additional users.
   1. **Project Description**: Write a **description** for the project to provide more context or details.
1. **Custom Fields**: If needed, you can also add **custom fields** specific to the project.
