### **How to Enable and Disable Project Features**
To enable or disable project features in 5day.io, follow these steps:

1. **Go to Project Settings**: First, navigate to the **Project Settings** of the project you want to configure.
### IMAGE
1. **Click on 'Features'**: Once in Project Settings, click on the **Features** tab.
1. **Enable or Disable Features**: In the features list, you'll see a range of features with toggle buttons next to each one. You can toggle **on** or **off** the features you want to enable or disable for that particular project.
   Here is a list of project features that you can enable or disable:
   - **Subtasks**: Divide large tasks into smaller, manageable parts.
   - **Priority**: Set task priority levels.
   - **Client**: Track work for a specific client at the project level.
   - **Multiple Assignee**: Assign tasks to multiple people.
   - **Tags**: Use custom tags to organize tasks.
   - **Project Finance**: Track billable, non-billable, and budget information.
   - **Time Tracking**: Track time for various types of work.
   - **Time Estimates**: Track time estimates for tasks.
   - **Time Approval**: Enforce manual approval for time entries.
   - **Overtime**: Track overtime hours in timesheets.
   - **Story Points**: Estimate the effort required to complete tasks.
   - **Assignee Wise Estimation**: Divide estimated task time among multiple assignees.
   - **Custom Fields**: Add custom fields to tasks for additional information.
   - **Multiple Task Types**: Assign and associate multiple task types within a project.
   - **Sprint Management**: Divide your project into time-boxed sprints (e.g., 2-week periods) to focus on specific goals and track progress within each sprint.  
1. **Save Changes**: Once you've made the necessary modifications, your changes will be saved automatically.
