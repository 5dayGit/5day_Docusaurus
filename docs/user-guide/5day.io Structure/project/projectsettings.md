### **Project Settings**
**Overview**\
Project settings in 5day.io provide a central hub for managing and configuring all aspects of a project. From here, users can modify project information, sections, and configure important features, manage integrations, and access various other project-specific settings. 

**How to Access Project Settings**

To access the project settings:

1. **Click on the Project Name**: Start by selecting the project for which you want to modify the settings.
1. **Open the Three-Dot Menu**: Click on the three-dot menu next to the project name.
1. **Select 'Project Settings'**: From the dropdown menu, click on **Project Settings**.
1. **Modify Settings**: In the project settings menu, you can adjust various project configurations like sections, features, and integration settings.
1. **Save Changes**: Any changes made in the project settings are automatically saved. Once done, you can click the **Back** button to return to the list of all projects.

### **How to Edit Project Information**
To edit the project info in 5day.io, follow these steps:

1. **Go to Project Settings**: First, navigate to the **Project Settings** of the project you want to edit.
1. **Access Project Info**: In the Project Settings menu, find and click on **Project Info**.
1. **Edit Project Info**:
   1. **Project Name**: You can change the project name by clicking on the project info icon.
   1. **Project Icon**: Upload a custom image or select pre-defined icon and colour/theme for the project icon.
   1. **Start and Due Dates**: Set the **start date** and **due date** for the project.
   1. **Project Status**: Choose the current **status** of the project (e.g., New, In Progress, In Review and Done).
   1. **Priority**: Set the **priority** level of the project (e.g., High, Medium, Low).
   1. **Project Prefix**: Select a **prefix** for the project, which will be used for task IDs.
   1. **Project Owner**: Assign the **owner** for the project. If there are multiple owners, click **Add Owner** and add additional users.
   1. **Project Description**: Write a **description** for the project to provide more context or details.
1. **Custom Fields**: If needed, you can also add **custom fields** specific to the project.

All changes made in the project info section will be saved automatically. Once you’re done, you can return to the project list by clicking the **Go Back** button.

### **How to Enable and Disable Project Features**
To enable or disable project features in 5day.io, follow these steps:

1. **Go to Project Settings**: First, navigate to the **Project Settings** of the project you want to configure.
1. **Click on 'Features'**: Once in Project Settings, click on the **Features** tab.
1. **Enable or Disable Features**: In the features list, you'll see a range of features with toggle buttons next to each one. You can toggle **on** or **off** the features you want to enable or disable for that particular project.
1. **Save Changes**: Once you've made the necessary modifications, your changes will be saved automatically.
1. **Go Back**: After you’re done, click the **Back** button to return to the list of projects.

#### **Available Features**
Here is a list of project features that you can enable or disable:

- **Subtasks**: Divide large tasks into smaller, manageable parts.
- **Priority**: Set task priority levels.
- **Client**: Track work for a specific client at the project level.
- **Multiple Assignee**: Assign tasks to multiple people.
- **Tags**: Use custom tags to organize tasks.
- **Project Finance**: Track billable, non-billable, and budget information.
- **Time Tracking**: Track time for various types of work.
- **Time Estimates**: Track time estimates for tasks.
- **Time Approval**: Enforce manual approval for time entries.
- **Overtime**: Track overtime hours in timesheets.
- **Story Points**: Estimate the effort required to complete tasks.
- **Assignee Wise Estimation**: Divide estimated task time among multiple assignees.
- **Custom Fields**: Add custom fields to tasks for additional information.
- **Multiple Task Types**: Assign and associate multiple task types within a project.
- **Sprint Management**: Divide your project into time-boxed sprints (e.g., 2-week periods) to focus on specific goals and track progress within each sprint.
