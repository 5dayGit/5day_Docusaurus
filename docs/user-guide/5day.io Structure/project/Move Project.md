### How to Move a Project 
To move a project in 5day.io, follow these simple steps: 
1. **Go to the Project**: Start by navigating to the project that you want to move. 
2. **Click on the Three-Dot Menu**: Once inside the project, click on the three-dot menu next to the project name. 
3. **Select 'Move to'**: In the dropdown menu, click on the **Move to** option. 
4. **Choose the Destination Space**: A prompt labeled **Where** will appear. Here, you can select the space or workspace where you want to move the project. 
5. **Confirm the Move**: Once you’ve selected the appropriate space or workspace, click on the **Move project** button. 
6. **Project Moved**: Your project will be automatically moved from its current space to the new one.
