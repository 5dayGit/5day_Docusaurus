# Rename Project

To rename a project in 5day.io, follow these simple steps:

1.  **Select the Project**: Click on the project name that you want to
    rename.
2.  **Open the Three-Dot Menu**: From the project view, click on the
    three-dot menu next to the project name.
    ### IMAGE
3.  **Click on \'Rename Project\'**: In the dropdown menu, select
    **Rename Project**.
4.  **Edit the Project Name**: The project name will become editable.
    Enter the new name for the project.
5.  **Changes Saved Automatically**: Once you've typed in the new name,
    the changes will be saved automatically.
