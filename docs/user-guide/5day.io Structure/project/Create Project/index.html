Create Project

You can create a project in two ways in 5day.io:

1. Create From Scratch​

Use this option to manually create a new project from any of the following locations:

Where can you add a project from?

  • Workspace Menu: Click on the workspace name in the left sidebar. Then click the three-dot menu (⋮) next to it and choose Add Project to add it under that workspace.
  • Space Menu: Inside a workspace, click on the space name, open the three-dot menu, and choose Add Project to add it under that space.
  • Project Menu: Next to an existing project, click the three-dot menu (⋮) and select Add Project. This will place the new project in the same space as the existing one.
  • Quick Add Button: From the top left navigation bar, click on the Quick Add (+) icon — represented by a plus sign inside a circle. Once the quick add menu opens, click on Project to start creating a new project.
  • Add (+) Button: A cross spaces and workspaces, you’ll find an Add (+) button.
    Click it, and from the dropdown menu, select Project to start project creation.

Tip: The location you choose will pre-fill the workspace/space in the project form automatically.

Steps to complete project creation:

  1. Choose Set a New Project
  2. Fill in the required fields:
    1. Project Name – The name of your project
    2. Where – (Auto-filled based on where you clicked, but editable)
    3. Description – (Optional) A short summary
    4. Duration – Start Date and Due Date
  3. Click Add to create your project

2. Create a Project Using a Template​

Use this option when you want to save time by creating a project with predefined structure, tasks, and sections.

Where can you add a project via template from?

  • Workspace Menu: Click on the workspace name in the left sidebar. Then click the three-dot menu (⋮) next to it and choose Add Project to add it under that workspace.
  • Space Menu: Inside a workspace, click on the space name, open the three-dot menu, and choose Add Project to add it under that space.
  • Project Menu: Next to an existing project, click the three-dot menu (⋮) and select Add Project. This will place the new project in the same space as the existing one.
  • Quick Add Button: From the top left navigation bar, click on the Quick Add (+) icon — represented by a plus sign inside a circle. Once the quick add menu opens, click on Project to start creating a new project.
  • Add (+) Button: A cross spaces and workspaces, you’ll find an Add (+) button.   Click it, and from the dropdown menu, select Project to start project creation.

Tip: The location you choose will pre-fill the workspace/space in the project form automatically.

Steps to Create From a Template​

  1. Click New Project from any of the locations above.
  2. In the project creation form, select the Use Template tab.
  3. Choose between:
    1. Default Templates – Templates provided by 5day.io for common use cases.
    2. Custom Templates – Templates previously saved by your team.
  4. Click on a template to preview and select it.
  5. Fill in the rest of the details:
    1. Project Name
    2. Where – Workspace or space (auto-filled, but editable)
    3. Description (optional)
    4. Start Date & Due Date
  6. Click Add to create the project.
