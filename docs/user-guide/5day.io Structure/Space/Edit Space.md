# Edit Space
Once a Space is created in 5day.io, users with the appropriate permissions can edit the Space to configure its structure, and settings.

Editing a Space helps maintain the accuracy, flexibility, and scalability of your Space organization.

## Permission Set


## Edit Space
There are two main ways to open the Edit Space panel

| Method | Navigation Path | Description |
|--|--|--|
| Space Settings | Workspace -> Spaces -> List of Spaces -> Edit | Edit from the central list of all Spaces within a Workspace |
| Space View | Space List or Board View -> More Menu (â‹®) beside Space name | Quick access for directly editing a specific Space |

## Space Settings

When you access the Edit option, you can perform the following actions depending on your permissions:

**Update Space Details**: Modify the Space name, icon, or color.

**Add Project**: Add a new Project within the Space. For more details refer Add Project

**Delete Space:** Delete selected space. For more details refer [Delete Space](https://github.com/5dayGit/5day_Docusaurus/blob/main/docs/user-guide/5day.io%20Structure/Space/Delete%20Space.md)
