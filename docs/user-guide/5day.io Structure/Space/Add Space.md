# Add Space
Space is the second level of organization within a Workspace. It allows you to group related Projects, Sections, Tasks, and Subtasks, making it easier to manage specific departments, teams, or focus areas under a single Workspace.

## Subscription Association
| Plan | Number of Space|
|--|--|
| Free | 5 |
| Growth | Unlimited |

## Add Space
You can create a new space in multiple ways within 5day.io:

### Create space via Quick Add

 1. From top/left navigation bar, click on 'Quick Add'
 2. Select 'Space' from left bar.
 3. Create a space.

### Create space via Workspace Settings

 1. You should have a permission to access workspace settings.
 2. From left navigation bar, access your workspace -> **workspace settings**.  
 3. Go to Spaces and Project.
 5. Click 'Add' to create a space.

### Create space via Views

 1. Go to any of the views (List/Board View) in your account. 
	 - Workspace Views
	 - Space Views
	 - Project Views
	 - Section Views 
 2. Click on 'Add' button and create a space.

## Space Details
| Field | Description |
|--|--|
| Icon & Color | Choose a representative icon and color to visually distinguish your space|
| Name | Provide a clear and meaningful name of your  space.|

The newly created Space will be visible in the Space List view within its parent Workspace. 
You can manage, update, or delete the Space based on your assigned role and permissions.

## Space ID
Once the space gets created, each will have a its unique ID to access directly via URL.
