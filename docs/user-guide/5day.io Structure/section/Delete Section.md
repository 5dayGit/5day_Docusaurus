## **How to Delete a Section**
In 5day.io, sections can be deleted easily using two different methods. Here’s how you can do it:
### **Option 1: Via the Project View**
1. **Open the Project**: Click on the project where the section is located.
2. **View Section List**: A list of all sections within the project will appear.
3. **Open the Three-Dot Menu**: Next to the section you want to delete, click on the **three-dot menu** (⋮).
4. **Click 'Delete'**: Select **Delete** from the menu.
5. **Confirm Deletion**: A confirmation box will appear. Click **Yes** to confirm and delete the section.

### **Option 2: Via Expanding the Project**
1. **Expand the Project**: Click the **arrow icon** next to the project name to expand and view all sections.
2. **Open the Three-Dot Menu**: Next to the section you want to delete, click on the **three-dot menu** (⋮).
3. **Click 'Delete'**: Select **Delete** from the menu.
4. **Confirm Deletion**: A confirmation box will appear. Click **Yes** to confirm the deletion.
