# Create Section
You can create a Section in a project using two easy methods:
### **Option 1: Create via Project Menu**
1. Go to the project where you want to create a section.
1. Click the **three-dot menu (â‹®)** next to the project name.

<img width="3560" height="2300" alt="image" src="https://github.com/user-attachments/assets/89120244-1871-47c5-9ce0-2d3394fd790f" />

1. Select **Add Section**.
1. In the **Add Section** form:
   1. **Project**: By default, the current project is selected. You can change this if needed.
   1. **Section Name**: Enter a name for your section.
   1. **Section Icon**: Customize the icon by:
      1. Choosing a **Color/Theme**
      1. Selecting an I**con** from the provided icon set
1. Click **Add** to create your section.

<img width="3560" height="2300" alt="image" src="https://github.com/user-attachments/assets/0b07e932-17db-40a5-944a-8d40d14cfb2b" />


### **Option 2: Create via Add Button**
1. Open the project where you want to add a section.
1. Click the **Add button (+)** located within the project workspace.
1. From the dropdown, select **Section**.
1. In the form that appears:
   1. **Project**: The current project will be selected by default (editable if needed).
   1. **Section Name**: Enter the section title.
   1. **Section Icon**: Customize by:
      1. Selecting a **Color**/T**heme**
      1. Choosing an **Icon** from the icon set
1. Click **Add** to complete the creation.
