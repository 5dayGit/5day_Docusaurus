Section Details

The Section Details in 5day.io provide a complete snapshot of a section’s key metrics and information.

Key Attributes

  • Status
    Displays the current status of the section:
    • New
    • In Progress
    • Completed
  • Total Tasks
    Shows the total number of tasks within the section.
  • Total Estimation
    Displays the combined estimated effort for all tasks in the section.
    • If task estimations (in hours) are added, they are shown here.
    • If Story Points are enabled (e.g., via Sprint Management or as a separate feature), the cumulative story points are also displayed.
    • Users can toggle between time and story points using the available icons.
  • Progress
    Shows the percentage of tasks completed, automatically calculated based on task completion status.
  • Remaining Work
    Displays the remaining effort:
    • Shown in hours if using time-based tracking.
    • Shown in story points if story points are enabled and selected.
  • Completed Work
    Shows the completed effort:
    • Displayed in hours or story points, depending on your selected view.
  • Description
    A free-text field where you can add notes, context, or any additional details about the section.

Note:

For Total Estimation, Remaining Work, and Completed Work, you’ll find icons that let you toggle between viewing data in hours (time) or story points. This flexibility helps teams monitor progress using the metric that best aligns with their workflow.

How to Access Section Details​

You can access Section Details in 5day.io in two simple ways:

Option 1: Using the 'Show Details' Button​

  1. Open the Project: Click on the project where the section is located.

  2. View Section List: You’ll see a list of all sections within the project.

  3. Click 'Show Details': Next to each section name, there is a Show Details button.
    Click this button to open and view the section details.

Option 2: Using the Three-Dot Menu​

  1. Open the Project: Click on the project where you want to view section details.
  2. Expand the Section List: Use the arrow icon beside the project name to expand and view all sections.
  3. Access the Menu: Next to each section, you’ll see a three-dot menu (⋮).
  4. Click 'Section Details': From the menu, select Section Details to view detailed information for that section.
