## Overview

The Reminder feature allows users to create reminders for important events, tasks, and subtasks.

Reminders help users stay informed about upcoming deadlines and scheduled activities.

Users can create reminders for:

- Standalone events
- Existing tasks and subtasks
## **Where You Can Create a Reminder**
Reminders can be created from two locations:
### **Option 1: From the Left Sidebar**
1. Click the **Reminder icon** in the left menu bar.
1. The Reminder editor will open.
### **Option 2: From My Reminders**
1. Navigate to **My Reminders** from the left sidebar.
1. Click **+ Add**.
1. The Reminder editor will open.
## **Reminder Editor**
When the Reminder editor opens:

- A default reminder icon is selected.
- You can change the icon by clicking the **pencil icon**.
- You must select a **Type**: 
  - Event
  - Tasks/Subtasks
# **Create an Event Reminder**
Select **Event** when the reminder is not linked to any task or subtask.
### **Steps**
1. Under **Type**, select **Event**.
1. Enter the **Title**.
1. (Optional) Add a **Description**.
1. Select the **Date**.
1. Select the **Time**: 
   1. Enter a custom time manually, or
   1. Choose from preset time options: 
      1. 11:00 AM
      1. 01:00 PM
      1. 06:00 PM
      1. 09:00 PM
1. Click **Create**.

The reminder will be listed under **My Reminders**.
# **Create a Task or Subtask Reminder**
Select **Tasks/Subtasks** when the reminder is linked to existing tasks.
### **Steps**
1. Under **Type**, select **Tasks/Subtasks**.
1. The **Title field converts into a searchable task selector**.
1. You can: 
   1. Search for a task or subtask by typing,
   1. Scroll and select from the list,
   1. Select multiple tasks or subtasks.
1. (Optional) Add a **Description**.
1. Select the **Date option**: 
   1. On due date
   1. Before due date
   1. After due date
   1. Custom date
1. Select the **Time**: 
   1. Enter a custom time manually, or
   1. Choose from preset time options: 
      1. 11:00 AM
      1. 01:00 PM
      1. 06:00 PM
      1. 09:00 PM
1. Click **Create**.

The reminder will be associated with the selected task(s) or subtask(s).
## **Viewing Reminders**
All created reminders are displayed in **My Reminders**.

If no reminders are created, the empty state screen will be displayed.
