# List of Subtasks
List of Subtasks displays every Subtask linked to a parent Task in a clear, expandable view - so you can track progress, assign owner, and manage small steps without leaving the main Task.

## Permission  
You should have ** View Permission** to view all the subtasks associated to a task.

## Access List of Subtasks 

### Task Details

-   Open any Task -> **Task Details** panel (right side)
-   Click the **Subtasks** tab
-   **All subtasks** appear in a full list -> **regardless of status** (New, In Progress, Done)
-   Best for deep management: add, reorder, bulk edit, or open full subtask details.

### List View

Each Task row shows a **Subtask icon**; when it has subtasks associated.
-   Click the icon -> **expand** to show the **List of Subtasks**
-   To include/exclude **Completed** subtasks: -> Click **three dots** (top-right of the view) -> turn on **Show completed subtasks**

### Board View

-   Each Task card shows a **Subtask icon**; when it has subtasks associated.
-   Click the icon -> **expand** to show the **List of Subtasks**
-   To include/exclude **Completed** subtasks: -> Click **three dots** (top-right of the view) -> turn on **Show completed subtasks**

### Show Subtasks as Separate Tasks (List View/Board View Settings)

-   If **"Show subtasks as separate tasks"** is enabled in Project/section views settings (three dots beside group by) -> All subtasks appear as **independent rows/cards** in **List View** and **Board View** -> No need to expand â€” they live alongside regular Tasks -> Still linked to parent 
-   **Enable it**: List/Board View Settings -> More Icon (three dots beside group by) -> toggle **Show subtasks as separate tasks**
