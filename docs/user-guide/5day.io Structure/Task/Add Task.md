# Add Task
In 5day.io, Add Task lets you instantly create a new unit of work within any project. 
Simply click + Add Task, enter a clear title, and optionally set assignee, due date, priority, and labels. Your Task is live in seconds - ready for Subtasks, tracking, and team collaboration.

## Subscription Association

## Who can create Task

Users must have **Add Task** permission to add new task or subtask.

If not allowed, the system will restrict task creation actions. Refer [Roles and Permission](https://github.com/5dayGit/5day_Docusaurus/blob/main/docs/user-guide/5day.io%20Structure/Workspace/Roles%20and%20Permission%20for%20Workspace.md) for more details.

## Ways to Add Task

5day.io offers multiple flexible ways to add tasks:

### Quick Add

Quick Add is a fast-access creation method available from the **primary navigation panel**.

**Steps** to create task

 1. Click on the **Quick Add (+)** icon.
 2. Select **Task**.
 3. Fill in the required fields: Task Name and Location
 4. Fill optional details if needed (Priority, Tags, Assignees, etc.).
 5. Click on Save.

### List View - Add Task

 1. Navigate to a project or section's **List View**.
 2. Click **Add Task** at the top or within a group.
 3. Provide a **Task Title** and press **Enter** or **Save**.
 4. Task is created immediately.

### Board View - Add Task

 1. Navigate to a project or section's **Board View**.
 2. Select the column (default - status) you want to add a task to.
 3. Click **+ Add Task**
 4. Provide a **Task Title** and press **Enter** or **Save**.
 5. Task is created immediately.

### Add Button from Project/Section

 1. Within a project or section page, use the standard **Add** button.
 2. This method works the same way as **Quick Add**.

### Import Tasks

 1. Tasks can be created in bulk by **Importing** an Excel file.
 2. Navigate to the Import Tasks option.
 3. Upload the formatted file and map fields as needed.
 4. Refer to the **Import Task Help Guide** for step-by-step instructions.

### My Work

 1. Tasks can also be created directly from the **My Work** page.
 2. Click **Add Button** and follow the same steps (Title, Location).
