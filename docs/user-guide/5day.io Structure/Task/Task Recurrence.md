# Task Recurrence
Task Recurrence turns any Task into a self-repeating machine - automatically creating new copies on your exact schedule (daily standups, weekly reports, monthly invoices, yearly audits) without lifting a finger after setup.

## Permission 
 
You need **Edit task permission** in the source task and **Add task permission** to create recurred tasks.

## Edit Recurred Task (Updates All Future Tasks)

-   Open **any** instance -> click **three dots** -> **Edit Recurrence**
-   Change settings (frequency, fields to copy, etc.) -> **Save**
-   **All future instances** update instantly with new rules.
-   **Past instances** remain unchanged (history preserved).

## Delete Recurred Tasks (Cancels All Future Tasks)

-   Open **any** instance -> **three dots** -> **Delete Recurrence**
-   Confirm â†’ **entire series stops**.
-   **All future Tasks are deleted**.
-   **Past Tasks stay** with full history and data.
