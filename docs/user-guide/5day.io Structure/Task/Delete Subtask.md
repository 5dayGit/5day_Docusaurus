# Delete Subtask
Deleting a subtask in 5day.io allows users to permanently remove unwanted or incorrect records from their project.

Deletion ensures that the subtask is no longer accessible or recoverable once removed. This action is irreversible, so it should be performed cautiously.

Both tasks and subtasks have similar deletion processes and are governed by permission settings..

## Who Can Delete Subtask

Deleting tasks depends on the **permission set** assigned to the user:

-   **Delete All:** User can delete any subtask across the project.
-   **Delete Owned:** User can delete only subtasks where they are assigned.
-   **Delete Related:** User can delete subtasks where they are a project owner or member.
-   **Delete None:** User is restricted from deleting any subtask.

## Delete Subtask

There are multiple ways to delete a **Subtask**:

### From List View / Board View

 1. Hover over a subtask row/card.
 2. Click on the **three dots (More option)** icon.
 3. A dropdown menu will appear.
 4. Click **Delete** from the menu.
 5. Confirm deletion to permanently remove the subtask.

### From Subtask Details Page

 1. Open the subtask you want to delete.
 2. On the **top-right corner** of the Subtask Details page, click the **three dots (More menu)**.
 3. Select **Delete** from the options.
 4. Confirm deletion to complete the action.

### From Subtask List Page

 1. Navigate to: **Task Details → Subtasks Tab → Subtask List**.
 2. Locate the subtask you want to delete.
 3. Click the **More option (three dots)** beside the subtask.
 4. Select **Delete Subtask**.
 5. Confirm deletion to remove the subtask permanently.

**Quick Note**
Always double-check before deleting, as this action is irreversible.
