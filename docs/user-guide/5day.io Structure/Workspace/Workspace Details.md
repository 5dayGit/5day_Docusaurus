# Workspace Details

**Workspace Details** is the one-stop screen to view and update everything that defines your Workspace - from its name and look to who’s in charge. It keeps your Workspace identity sharp and ownership crystal-clear.

## Who can Manage

You must have **Edit Workspace** permission to make changes here.

## Access Workspace Details

1.  Go to you workspace -> click on more icon (three dots) → **Workspace Settings** → **General**
2.  You’re now inside **Workspace Details**

## Workspace Details

| Field	 | Description |
|--|--|
| Icon and Color | Pick any icon + color combo to make this Workspace stand out in the sidebar |
| Name | Rename the Workspace anytime |
| About | Add a short note about purpose, scope, or rules (optional but helpful) |
| Workspace Owner | View current owner’s photo, name, and email, refer [Workspace Owner](https://github.com/5dayGit/5day_Docusaurus/blob/main/docs/user-guide/5day.io%20Structure/Workspace/Workspace%20Owner.md) for more details. |
| Delete Workspace | Permanently remove the entire Workspace + everything inside, refer [Delete Workspace](https://github.com/5dayGit/5day_Docusaurus/blob/main/docs/user-guide/5day.io%20Structure/Workspace/Delete%20Workspace.md) for more details. |
