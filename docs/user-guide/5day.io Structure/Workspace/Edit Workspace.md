
# Edit Workspace
Once a workspace is created in 5day.io, users with the appropriate permissions can **edit the workspace** to configure its structure, settings, and users.

Editing a workspace helps maintain the accuracy, flexibility, and scalability of your workspace organization.

## Permission Set


## Edit Workspace
There are two main ways to open the Edit workspace

| Method | Navigation Path | Description |
|--|--|--|
| Account Settings | Account Settings -> List of Workspaces -> Edit | Edit from the central list of all workspaces |
| Workspace View | Workspace List or Board View → More Menu (⋮) beside workspace name | Quick access for directly editing a specific workspace |

## Workspace Settings

When you access the Edit option, you can perform the following actions depending on your permissions:

**Add Space**: Add a new space under the workspace.

**Add Project**: Add a new project directly within the workspace. Refer: _Add Project Guide_

**Add/Invite People**: Add/Invite team members to the workspace. Refer: _User & Roles Guide_

**Rename**: Change the name of the workspace.

**Workspace Settings**: Access detailed workspace configurations (explained below).

| Pages | Description |
|--|--|
| **General Info** | **Manage workspace's identity and ownership** |
| Icon & Color | Choose a visual icon and color theme for easy identification. |
| Name | The display name of your workspace. |
| Description | A short explanation about the purpose or scope of the workspace. |
| Owner | The current owner responsible for workspace administration. |
| **Spaces & Projects** | View and manage all spaces and projects under this workspace. Change their order, navigate, and access related settings. |
| **Timesheet Settings** | Configure time tracking rules, settings, and behavior for this workspace. Refer: _Timesheet Module_ for more. |
| **Customization** |  |
| Status Workflow | Define the workflow stages of tasks. Refer: _Status Workflow Guide_ |
| Task Types | Create and manage types of tasks for better categorization. Refer: _Task Types Guide_ |
| Priorities | Define and manage priority levels for tasks. Refer: _Priority Settings Guide_ |
| Sprint Management | Configure sprint tracking and agile settings. Refer: _Sprint Management Guide_ |
| Integrations | The Integration Centre allows to connect the workspace with third-party apps, tools, or services for seamless operations and enhanced functionality. Explore available integrations based on your plan |
