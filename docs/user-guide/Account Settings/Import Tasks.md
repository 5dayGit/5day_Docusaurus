## **How to Import Tasks**
In 5day.io, you can import tasks in bulk to any project using a structured Excel template.\
This feature is especially useful when migrating tasks from spreadsheets or other tools.

### **Steps to Import Tasks (Account Level)**
1. **Go to Account Settings**
   1. Click on **Data Import History** from the sidebar.
1. **Click + Import**
   1. A confirmation box will open.
1. **Choose Import Type and Project**
   1. Select **Task** as the import type.
   1. Choose the **Project** where you want to import the tasks.
1. **Download the Sample File**
   1. Click **Download Sample File**.
   1. The Excel file includes two sheets:
      1. **Template**: Where you’ll enter your task data (e.g., title, description, due date, assignee).
      1. **Guidelines**: Reference sheet explaining each column in the template.
1. **Fill Out the Template File**
   1. Open the file and input your tasks in the **Template** sheet.
   1. Use the **Guidelines** sheet for field formats and required values.

1. **Upload and Import the File**
   1. Once your file is ready, return to the import window in 5day.io.
   1. In the import popup, click **Upload from computer** and select your filled Excel file.
   1. Click **Import** to start the import process.

### **After the Import**
Once the file is processed, you’ll see it listed in the **Data Import History** table.

You’ll be able to view:

- **Date and Time of import**
- **File Name**
- **Imported By**
- **Import Type** (Task)
- **Status**: (Completed or Failed)
- **Total Records**, **Successful Imports**, and **Failed Records**

If any records failed:

- You can view the number of failed entries.
- Download the **error file** to see what went wrong and correct it.
- Re-upload the corrected file if needed.
