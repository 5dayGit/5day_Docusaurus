Create a Work Schedule​

In 5day.io, you can create custom work schedules to define working hours, hybrid work models, and holidays for your team. You can either create a new schedule from scratch or duplicate an existing one to save time.

1. Create a New Work Schedule​

  1. Go to Work Schedule:\

    IMAGE​

    Navigate to Account Settings → General → Work Schedule.
  2. Click on Add:
    Inside the Work Schedule section, click the Add button next to the list of existing work schedules to start creating a new one.
  3. Enter Work Schedule Name:\

    IMAGE​

    Give your new work schedule a meaningful name (e.g., "Regular Working Hours" or "US Hybrid Team").
  4. Set as Default (Optional):
    Enable this option if you want this schedule to become the organization's default working schedule.
  5. Define Work Days and Settings:
    1. Start of Calendar Week: Choose between Monday or Sunday.
    2. Enable Hybrid Work:
      1. Assign work location (Home or Office) per day.
      2. Optionally allow members to update their hybrid working status manually.
    3. Enable or Disable Working Days:
      Select the days that are active (e.g., Monday–Friday).
    4. Set Start and End Times:
      Define working hours for each active day.
    5. Set Daily Location (if Hybrid is Enabled):
      Choose Office or Home for each working day.
  6. Add Holidays:
    1. Switch to the Holidays tab.
    2. Click Add Holiday, and define the Holiday Name and Holiday Date.
  7. Auto-Save:
    As you set up your work schedule, all changes are automatically saved.
    No need to manually click any Save button.

2. Duplicate an Existing Work Schedule​

If you already have a work schedule similar to the one you need, you can simply duplicate it and make changes instead of creating a new one from scratch.

Here’s how:

  1. Go to Work Schedule:
    Navigate to Account Settings → General → Work Schedule.
  2. Find the Schedule to Duplicate:
    Locate the existing work schedule you want to copy.
  3. Click the Three-Dot Menu (⋮):\

    IMAGE​

    Next to the schedule name, click on the three dots.
  4. Select Duplicate:
    Click on the Duplicate option from the menu.
  5. Edit the Duplicated Work Schedule:
    1. The duplicated schedule will appear immediately.
    2. You can modify the work days, hours, hybrid settings, and holidays as needed.
    3. Update the Work Schedule Name to reflect the new schedule.

All edits are automatically saved as you make changes.
