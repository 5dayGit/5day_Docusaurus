# **How to Import People**
In 5day.io, you can import multiple people in bulk at the account level using a structured Excel template.\
This is especially useful when onboarding an entire team or migrating users from another platform.

### **Steps to Import People (Account Level)**
1. **Go to Account Settings**
   1. Navigate to the top menu and click **Account Settings**.
1. **Open Data Import History**
   1. From the sidebar, click on **Data Import History**.
         ### IMAGE
1. **Click + Import**
   1. In the top-right, click the **+ Import** button to start the import process.
1. **Select 'People' as Import Type**
   1. A confirmation box will appear.
   1. Choose **People** as the data type you want to import.
1. **Download the Sample Template**
   1. Click **Download Sample File**.
   1. The Excel file will contain:
      1. A **Template** sheet where you’ll enter the people data (e.g., name, email, role).
      1. A **Guidelines** sheet explaining the format and allowed values.
1. **Fill Out the Template File**
   1. Enter the necessary details in the **Template** sheet.
   1. Refer to the **Guidelines** sheet to ensure accuracy and completeness.
1. **Upload and Import the File**
   1. Once your file is ready, **return to the import window** in 5day.io.
   1. Click **Upload from computer**, select your completed Excel file, and click **Import**.

### **After the Import**
Once the import is complete, the import record will appear in the **Data Import History** table with key information such as:
   ### IMAGE
- **Date and Time**
- **Imported By**
- **Import Type** (People)
- **Status** (Completed or Failed)
- **Total Records**
- **Successful Imports**
- **Failed Records**

If there are failed records:

- You’ll see an **error icon** with the number of failed entries.
- Click the icon to **download the failure file**, which includes error messages explaining why each record failed.
- You can correct the file and re-import the failed records.
