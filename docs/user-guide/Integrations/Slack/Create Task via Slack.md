# Create Task via Slack
Create 5day.io tasks directly from Slack - from any message, channel, or DM. 
Tasks include full context: message text, attachments, sender, and a link back to Slack.

## Prerequisites
1.  Your Slack account must be connected via the 5day.io bot.
2.  You must have Create Task permission in the target 5day.io Workspace/Space/Project.

## Create Task

### From a Channel
-   In any **channel**, click the **⚡ (Shortcuts)** icon in the message field.
-   Search for **5day.io** -> **Create Task**.
-   Task form opens with **channel name** in context.
-   Fill details and click **Create**.

### From a Group DM
-   In a **group DM**, click the **⚡ (Shortcuts)** icon.
-   Select **5day.io-create-task**.
-   Click **Create**.

### From a Direct Message
-   In a **DM** (with bot or teammate), use **⚡ Shortcuts** -> **5day.io-create-task**.
-   Fill data and create task.

### From Any Message (Right-Click)
-   **Hover** over a message -> click **More actions (⋯)**.
-   Choose Create task in 5day.io
-   Message text = title.
-   Edit and Create.

### Using the /5day.io Slash Command
-   Type directly in any chat or channel: /5day.io-create-task where 5day.io is added as a bot.

## Task Fields You Can Set
| Field | Description |
|--|--|
| Project | Select a project where you would like to create a task |
| Task | Provide a title of your task |
| Section | Select a section under which the task needs to be created |
| Description | Provide description of your task |
| Start Date| Add start date of your task |
| Due Date| Add due date for your task |
| Assignee| Select an assignee whom you would like to assign a task. |

**Quick Notes**

-   **Use message context**: Always create tasks **from the message** to auto-capture quotes.
-   **Task link in Slack**: After creation, a **task preview card** appears in the chat with a direct link to 5day.io.
