# List of Commands 

| Command | Description |
|--|--|
| `/5day.io-create-task` | Create a new task in 5day.io. Supports inline parameters (title, assignee, due date, etc.). |
| `/5day.io-disconnect`| Disconnect your slack account from 5day.io. Stops all notifications and task creation. |
| `/5day.io-help`| Display the full list of supported commands with brief descriptions. |

## Other References

To connect your slack account with 5day.io, refer [Connect slack account](https://help.5day.io/docs/user-guide/Integrations/Slack/Connect%20slack%20account).

To create task via Slack, refer [Create task via Slack](https://help.5day.io/docs/user-guide/Integrations/Slack/Create%20Task%20via%20Slack)
