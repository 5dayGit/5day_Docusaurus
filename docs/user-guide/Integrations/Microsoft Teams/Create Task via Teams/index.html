Create Task via Microsoft Teams

Turn any Microsoft Teams conversation into an actionable task in 5day.io - without leaving Teams.

The create task command lets you capture ideas, assign ownership, set due dates, and add context instantly from channels, group chats, direct messages, or even a single message.

Once created, the task appears in your 5day.io workspace with a link back to the original Teams message, ensuring full traceability.

Prerequisites​

  1. Your Teams account must be connected via the 5day.io bot.
  2. You must have Create Task permission in the target 5day.io Workspace/Space/Project.

Create Task​

From a Channel​

  • Open the desired channel in Microsoft Teams.
  • Click the … (More actions) menu below the message input box.
  • Select 5day.io -> Create Task.
  • A task form opens in a pop-up.
  • Fill in details (title auto-fills from channel name if blank).
  • Click Create Task.

From a Group Chat​

  • In a group chat, click the … (More actions) in the top-right of the chat header.
  • Choose 5day.io -> Create Task.
  • The task form appears with chat name pre-filled as context.
  • Add assignees from the chat participants.
  • Click Create Task.

From a 1:1 or Direct Message​

  • Open your 1:1 chat with the 5day.io bot or any teammate.
  • Click the … (More actions) below the message box.
  • Select 5day.io -> Create Task.
  • The task form opens with "Private Task from DM" suggested.
  • Click Create Task.

From Any Message (Right-Click)​

  • Hover over any message (yours or someone else’s).
  • Click the … (ellipsis) on the message.
  • Select More actions -> 5day.io: Create Task.
  • The message text becomes the task title.
  • Edit fields and click Create Task.

Using the @create task Command​

  • Type directly in any chat or channel: @5day.io create task where 5day.io is added as a bot.

Task Fields You Can Set​

FieldDescription
ProjectSelect a project where you would like to create a task
TaskProvide a title of your task
SectionSelect a section under which the task needs to be created
DescriptionProvide description of your task
Start DateAdd start date of your task
Due DateAdd due date for your task
AssigneeSelect an assignee whom you would like to assign a task.

Quick Notes

  • Use message context: Always create tasks from the message to auto-capture quotes.
  • Task link in Teams: After creation, a task preview card appears in the chat with a direct link to 5day.io.
