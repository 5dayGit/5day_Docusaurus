# List of Commands 
| Command | Description |
|--|--|
| `connect`| Connect your Microsoft Teams account to your 5day.io account. Starts the authorization flow. |
| `disconnect`| Disconnect your Teams account from 5day.io. Stops all notifications and task creation. |
| `create task` | Create a new task in 5day.io. Supports inline parameters (title, assignee, due date, etc.). |
| `help`| Display the full list of supported commands with brief descriptions. |
| `support`| Get direct links to the Teams Integration Help Center and contact support. |
