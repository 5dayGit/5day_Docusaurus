## Overview
Once Google Calendar is connected and a calendar is selected, your events and tasks automatically sync between Google Calendar and [5day.io](http://5day.io/).

You can view and manage everything from **My Calendar**.

## **How to View Synced Google Calendar Events**
1. Log in to [5day.io](http://5day.io/).
1. Click the **Home icon**.
1. Click **My Calendar**.
1. Click the **Syncing** option in the top-right corner.
1. Select the calendar(s) you want to display.

You can select one or multiple calendars.

All selected calendar events will appear in **My Calendar**.

## **What Gets Synced from Google Calendar**
- All events from the selected Google Calendar are shown in **My Calendar**.
- Past and upcoming events are displayed.
- If an event is deleted in Google Calendar, it is removed from My Calendar.
- If the integration is disconnected, all synced events disappear from My Calendar.

You can click on any Google Calendar event to:

- View event details (title, date, time)
- Open the event in Google Calendar using the external link icon

## **How Tasks Sync to Google Calendar**
Tasks sync automatically when:

- The task has a due date
- The task is assigned to you
- Google Calendar integration is enabled

You can create a task:

- From the **Add Task** button in My Calendar
- From anywhere else in [5day.io](http://5day.io/)

Once created, the task appears in:

- **My Calendar**
- Your selected Google Calendar

## **What Happens When Tasks Are Updated or Deleted**
- If a task is updated in [5day.io](http://5day.io/), the changes reflect in Google Calendar.
- If a task is deleted in [5day.io](http://5day.io/), it is removed from Google Calendar.
- Google Calendar meetings cannot be deleted from [5day.io](http://5day.io/).
