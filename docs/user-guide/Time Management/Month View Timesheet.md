### **Month View in Timesheet**
The **Month View** offers a high-level calendar layout, allowing users to quickly visualize all time entries logged across an entire month.
#### **What You Can Do in Month View**
- **Calendar Layout**: Displays the selected month in a calendar format, with dates and weekdays clearly marked.
- **See Logged Entries per Day**:
  - For each date, users can view a summary of all time entries logged.
  - Each entry displays the task name and duration.
- **Navigate Between Months**:
  - Navigate to past or upcoming weeks using the left/right arrow buttons at the top.
- **User Switcher (Showing For)**:
  - Members will only see their own logged entries.
  - Admins, Project Managers, or users with proper permissions can select and view entries for other team members.
- **Summary Panel**:
  - A summary at the bottom will display detailed time entry summary.
- **Search & Filter Options**:
  - Search: Look up time entries using task or subtask names, task or subtask IDs, or by people.
  - Filter entries by **Budget Type** (e.g., Billable / Non-Billable) to refine the view.
